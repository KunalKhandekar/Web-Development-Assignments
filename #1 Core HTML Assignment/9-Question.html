HTML Text Formatting.

  1. bold (b)
    • Used in HTML to make text bold.
    • It is a simple and straightforward way to emphasize content visually.
    • However, it is recommended to use CSS styles for better control over text formatting.
  2. italic (i)
    • Used in HTML is used to apply an italic style to text.
    • It is commonly used to indicate emphasis or to distinguish a specific portion of text.
    • This tag is a presentational element and should be used for stylistic purposes only, not for semantic meaning.
  3. emphasize (em)
    • Used in HTML is used to emphasize text with semantic meaning.
    • It is commonly used to highlight words or phrases that carry importance or significance in the context of the content.
    • It is typically displayed in italics by default, although the exact styling can vary based on the browser and CSS applied.
  4. strong
    • Used HTML is used to indicate strong importance or emphasis on text.
    • It is similar to the <b> tag, but the <strong> tag carries semantic meaning rather than just visual styling.
    • Also enhance accessibility, as it provides cues to screen readers and search engines about the importance of the enclosed text.
  5. mark
    • Uesd in HTML to highlight or mark specific text within a document.
    • It visually highlights the enclosed text with a background color or other distinct styling.
    • It is widely supported by modern browsers and contributes to enhancing the visual presentation and readability of the marked text.
